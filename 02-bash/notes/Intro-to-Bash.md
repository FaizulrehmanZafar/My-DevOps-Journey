What is Bash Scripting? –

Bash: a command-line tool to interact with your computer
Bash Script: A file containing a series of commands you want the computer to execute automatically

The Shebang Line –

-Plays crucial role in execution of bash scripts.

Comments –

-To write a comment in a script you type ‘#’ and anything written past the # will be treated as a comment
-You can prevent commands from being executed by adding the # at the start of the line as then it will be recognised as a comment
